## 프로젝트 소개
금오공과대학교 학생을 대상으로 웹 개발 부트캠프 WebKIT640을 홍보&모집하는 홈페이지 입니다. 다양한 모집 정보와 학생 후기를 확인할 수 있습니다. 모집 기간일 경우 회원가입을 통해 부트캠프 신청이 가능합니다. 2022년 12월 2기 모집을 시작으로 현재까지 서비스되고 있습니다.

<br/>
